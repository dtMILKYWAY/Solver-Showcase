## 🎁 新增功能

### 可复用组件
```tsx
import { ButterBear, AnimatedButterBear } from './components/ButterBear'

// 静态小熊
<ButterBear size={80} />

// 动画小熊（带光晕）
<AnimatedButterBear size={120} />
```

### 自定义属性
```tsx
<ButterBear
  size={80}           // 尺寸
  className=""       // 自定义 class
  animated={true}    // 是否启用浮动动画
/>
```

## 💡 使用真实 GIF 图片

如果想用真实的小熊 GIF 图片：

```tsx
// 1. 将 GIF 放入 public/images/bear.gif
// 2. 创建组件：
<img
  src="/images/bear.gif"
  alt="黄油小熊"
  className="w-20 h-20 animate-bounce"
/>

// 3. 或者在现有位置替换：
<ButterBear size={80} />
// 替换为：
<img src="/images/bear.gif" alt="黄油小熊" className="w-20 h-20" />
```
